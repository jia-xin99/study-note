          03 SQL优化

          （1）插入数据

          【1】使用insert

          • 场景：使用多条insert语句插入到同一数据库表。

          • 优化方法1：批量插入

            • insert 表名 values(xxx),(xxx),(xxx);

          • 优化方法2：使用事务

            • 先开启事务，然后一条条插入数据，最后提交。

          • 优化方法3：主键顺序插入

            • 按主键顺序插入，性能高于乱序插入

          【2】大批量插入数据

          • 使用MySQL的load指令
